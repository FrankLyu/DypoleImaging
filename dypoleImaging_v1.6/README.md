# ImageAnalyze
ImageAnalyze software (since Oct. 2016)

Requirements:
- Python 3.6(.12)
- wxPython (4.0.4)
- pySpin (FLIR Spinnaker wrapper for Python)

On Ubuntu:
- Install the Python clr wrapper by doing `pip install -U git+https://github.com/pythonnet/pythonnet`
